# Opportunity Scoring Rubric — Detailed Criteria

## ServiceNow Alignment (Weight: 25%)

| Score | Criteria |
|-------|---------|
| 9-10 | Explicitly requires ServiceNow GRC/IRM/Audit development or implementation |
| 7-8 | Explicitly requires ServiceNow platform work (ITSM, HRSD, CSM, custom apps) |
| 5-6 | ServiceNow mentioned as one of several platforms; or GRC/compliance work where ServiceNow is the likely platform |
| 3-4 | IT modernization or workflow automation where ServiceNow could be proposed but isn't specified |
| 1-2 | General IT services; ServiceNow is a stretch |
| 0 | No ServiceNow relevance |

## GRC/Compliance Adjacency (Weight: 20%)

| Score | Criteria |
|-------|---------|
| 9-10 | Core GRC work: audit management, compliance frameworks (NIST 800-53, FISMA, FedRAMP), risk assessment, POA&M management |
| 7-8 | Compliance-adjacent: ATO support, security assessment, continuous monitoring, policy management |
| 5-6 | Touches compliance indirectly: system migration requiring ATO, IT governance, data governance |
| 3-4 | General security or risk work without specific GRC framework alignment |
| 1-2 | Tangential: general IT consulting at an agency with GRC needs |
| 0 | No compliance/GRC relevance |

## Delivery Model Fit (Weight: 15%)

| Score | Criteria |
|-------|---------|
| 9-10 | Deliverable-based or outcome-based; Vectis can deliver solo; scope matches capabilities exactly |
| 7-8 | Deliverable-based; Vectis delivers as sub with clear scope; timeline is realistic |
| 5-6 | Mixed model (some deliverable, some staff aug); subcontracting path exists |
| 3-4 | Primarily staff aug but with defined deliverables; would require on-site presence |
| 1-2 | Pure staff aug / body shop; long-term FTE replacement |
| 0 | Requires capabilities Vectis doesn't have and can't acquire |

## Contract Vehicle Access (Weight: 15%)

| Score | Criteria |
|-------|---------|
| 9-10 | Open solicitation accessible to small businesses; Vectis meets all requirements |
| 7-8 | Small business set-aside Vectis qualifies for; or known prime with existing relationship |
| 5-6 | Full & open with clear subcontracting path; or set-aside with teaming option |
| 3-4 | Restricted vehicle (GWAC, BPA) Vectis isn't on; subcontracting path unclear |
| 1-2 | Set-aside Vectis doesn't qualify for; no obvious teaming partner |
| 0 | Sole source to another vendor; or requires certifications Vectis lacks |

## NowForge Leverage (Weight: 10%)

| Score | Criteria |
|-------|---------|
| 9-10 | NowForge directly accelerates core deliverable (ServiceNow app build, schema design, GRC implementation) |
| 7-8 | NowForge accelerates significant portion of work (migration, configuration, testing) |
| 5-6 | NowForge provides some advantage (documentation, analysis, template generation) |
| 3-4 | Marginal NowForge relevance; mostly advisory/non-technical work |
| 1-2 | No NowForge application; purely non-ServiceNow technical work |
| 0 | Not applicable |

## Revenue Potential (Weight: 10%)

| Score | Criteria |
|-------|---------|
| 9-10 | $100K+ realistic Vectis revenue; multi-year potential; recurring |
| 7-8 | $50K-$100K realistic Vectis revenue; 6+ month engagement |
| 5-6 | $20K-$50K realistic Vectis revenue; defined scope |
| 3-4 | $5K-$20K or uncertain revenue; short engagement |
| 1-2 | < $5K or highly speculative revenue |
| 0 | No revenue path identified |

## Strategic Value (Weight: 5%)

| Score | Criteria |
|-------|---------|
| 9-10 | Marquee agency, builds past performance in target vertical, reference-account potential |
| 7-8 | Good agency, builds capabilities in priority area, teaming relationship potential |
| 5-6 | Decent portfolio addition; incremental capability building |
| 3-4 | Limited strategic value beyond revenue |
| 1-2 | Commodity work; no portfolio differentiation |
| 0 | Could damage positioning (e.g., pure body shop at a non-target agency) |
